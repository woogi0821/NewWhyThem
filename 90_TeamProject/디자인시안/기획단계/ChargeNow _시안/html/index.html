시간 예약

바로 충전

📋 일반 ChargeNow 앱 v2.5 업데이트 안내 2026.02.20 ▾

ChargeNow 앱이 v2.5로 업데이트되었습니다.

주요 변경사항:
· 즐겨찾기 충전소 최대 20개로 확대
· 충전 완료 예상 시간 표시 기능 추가
· 지도 마커 디자인 개선
· 기타 버그 수정 및 성능 개선

예약 예약 취소는 어떻게 하나요? ▾

내 예약 페이지에서 예약 취소가 가능합니다.
충전 시작 30분 전까지는 무료로 취소할 수 있으며,
이후 취소 시에는 예약 수수료가 발생할 수 있습니다.

예약 예약 시간에 늦으면 어떻게 되나요? ▾

예약 시간 이후 15분이 지나면 예약이 자동으로 취소됩니다.
늦을 것 같은 경우 미리 예약 시간을 변경하거나 취소 후 재예약하시길 권장합니다.

충전 급속 충전과 완속 충전의 차이가 뭔가요? ▾

급속 충전은 50kW~200kW 출력으로 약 20~40분 내 80%까지 충전 가능합니다.
완속 충전은 7kW 출력으로 완전 충전까지 6~8시간이 소요되지만 요금이 저렴합니다.
차량 배터리 수명을 위해 일상적인 이용 시에는 완속 충전을 권장합니다.

충전 충전 중에 오류가 발생했어요. 어떻게 해야 하나요? ▾

충전 중 오류 발생 시 아래 순서로 조치해주세요.

1. 충전 커넥터를 분리 후 재연결
2. 앱에서 충전 상태 새로고침
3. 오류가 지속되면 충전기 옆 QR코드를 통해 즉시 신고
4. 고객센터 1:1 문의로 접수 시 빠른 처리를 도와드립니다.

요금/결제 충전 요금은 어떻게 계산되나요? ▾

충전 요금은 실제 충전된 전력량(kWh)에 단가를 곱해 계산됩니다.

· 급속 50kW: 300원/kWh
· 급속 100kW 이상: 280원/kWh
· 완속 7kW: 240원/kWh

마이페이지의 충전 요금 계산기를 통해 예상 요금을 미리 확인하실 수 있습니다.

요금/결제 결제 수단은 어떤 것을 지원하나요? ▾

현재 지원하는 결제 수단입니다.

· 신용카드 / 체크카드 (국내 전 카드사)
· 카카오페이 / 네이버페이
· ChargeNow 포인트 (충전 완료 시 적립)

간편결제 등록 후에는 별도 인증 없이 자동 결제됩니다.

계정 비밀번호를 잊어버렸어요. ▾

로그인 화면 하단의 비밀번호 찾기를 클릭하세요.
가입 시 등록한 이메일 주소를 입력하면 재설정 링크가 발송됩니다.
이메일을 받지 못한 경우 스팸함을 확인하거나 1:1 문의로 접수해주세요.

계정 회원 탈퇴는 어떻게 하나요? ▾

마이페이지 → 프로필 수정 → 하단 회원 탈퇴 버튼을 통해 탈퇴하실 수 있습니다.

탈퇴 시 유의사항:
· 예약 중인 내역은 자동 취소됩니다
· 잔여 포인트는 즉시 소멸됩니다
· 탈퇴 후 30일 이내 재가입 불가
